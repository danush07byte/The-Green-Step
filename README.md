# The Green Step | Climate Action Platform
**Small Action Create Big Change – Be part of the Solution**

The Green Step is a community-driven climate awareness platform designed for the public in Malaysia. It aims to make climate action accessible, practical, and inspiring through educational content, community galleries, and direct engagement opportunities.

##  Project Structure
The website consists of 5 main pages, all fully responsive for Desktop and Mobile:

* **About Us (`index.html`)** - Mission, Vision, and Goals.
* **Gallery (`gallery.html`)** - Visual showcase of environmental campaigns.
* **Get Involved (`get-involved.html`)** - Volunteer registration and event list.
* **Contact (`contact.html`)** - Contact details and feedback form.
* **Donate (`donate.html`)** - Contribution system with custom selectors.

##  Technologies Used
* **HTML5** - Semantic structure.
* **CSS3** - Custom styling and Media Queries.
* **jQuery (v3.6.0)** - Animations and form logic.
* **Google Fonts** - Montserrat, Open Sans, and Poppins.

##  Key Features
### 1. Interactive Donation System
* **Dynamic Validation:** The "Donate" button activates only when an amount and designation are selected.
* **Custom Dropdowns:** Smooth MYR/USD and Campaign selection overlays.
* **Visual Ticks:** Selected options feature a green highlight and a checkmark (✔).

### 2. Responsive UI
* Separate layouts for **Desktop (1025px+)** and **Mobile (below 1024px)** using dedicated media queries to ensure the wireframe design is maintained on all screens.

### 3. Feedback Interactivity
* The contact form includes a decorative textarea where the internal lines change state when the user begins typing.

##  Setup Instructions
1. Open `index.html` in any browser.
2. Internet connection is required to load **jQuery** and **Google Fonts** via CDN.
3. Ensure all project images are located in the root folder.
